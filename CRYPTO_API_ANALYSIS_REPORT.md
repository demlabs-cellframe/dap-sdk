### Примеры использования обновленного API
```python
# Создание ключа (DILITHIUM по умолчанию)
key = DapCryptoKey()

# Одиночная подпись
signature = DapSign(data, pvt_key=key)
assert signature.verify(data, key)

# Композитная мульти-подпись (смешанные типы ключей)
mixed_keys = [DapCryptoKey(), DapCryptoKey(DapKeyType.FALCON)]
multi_sig = DapSign(data, pvt_keys=mixed_keys)
assert multi_sig.verify(data, mixed_keys)

# Агрегированная подпись (только CHIPMUNK)
chipmunk_keys = [DapCryptoKey(DapKeyType.CHIPMUNK) for _ in range(3)]
agg_sig = DapSign(data, pvt_keys=chipmunk_keys)
assert agg_sig.verify(data)  # Без явного указания ключей
```
